IDEA常用快捷键

发表于 2017-07-01 | 0 comments

  1. <Ctrl>+ <click> 定位到变量声明处

  2. <Ctrl> + <\B> 返回上一次浏览的位置

  3. <Ctrl> + <Alt> + <L> 自动对齐

  4. <Ctrl> + <F12> 显示类成员

  5. <Alt> + Left\Right 切换标签页

  6. <Alt> + <Enter> 光标在错误上调出自动错误处理列表

  7. <Ctrl> + <-> 折叠光标所在的方法

  8. <Ctrl> + <Shift> + <-> 折叠所有方法

  9. <Ctrl> + <=> 展开光标所在的方法

  10. <Ctrl> + <Shift> + <=> 展开所有方法
